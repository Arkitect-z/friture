Friture

Real-time audio spectrogram

Friture is a program designed to analyze audio input in real-time.

Features

Friture displays audio data in several widgets, such as a scope, a spectrum analyzer, or a rolling 2D spectrogram. Learn more about the features of Friture.

Audio and programming notes
