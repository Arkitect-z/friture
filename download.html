Other platforms

On other platforms (Mac OS, Linux), please use the PyPI package. Some dependencies have to be manually installed in this case. They are documented for Linux in the wiki (notes contributed by Lee McKusick) when installing from source. The same dependancies are likely to have to be installed when using the PyPI package too.

Bugs

If you encounter any problems while installing or running Friture, please report them.
