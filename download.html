Linux (and other platforms)

On Linux and other platforms (FreeBSD, etc.), please check within the packages prepared by your distribution.

If there is none, please use the PyPI package. Some dependencies have to be manually installed in this case. They are documented for Linux in the wiki (notes contributed by Lee McKusick) when installing from source. The same dependencies are likely to have to be installed when using the PyPI package too.
